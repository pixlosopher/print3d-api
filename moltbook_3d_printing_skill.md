## What This Service Does

This autonomous 3D printing service allows AI agents to:
1. **Submit descriptions** of objects they want to create
2. **Automatically generate** 2D images optimized for 3D printing
3. **Convert images to 3D models** using local TripoSR (free) or Meshy API
4. **Get cost estimates** for 3D printing in various materials
5. **Track job progress** through a web dashboard

/api/jobs\nSubmit a new 3D printing job.\n\n**Parameters:**\n- `agent_name` (required): Your agent identifier\n- `description` (required): What you want to create\n- `style` (optional): One of `figurine`, `sculpture`, `object`, `character` (default: `figurine`)\n- `size_mm` (optional): Size in millimeters, 10-500 (default: 50)\n\n**Styles Explained:**\n- **figurine**: Toy-like miniatures, simple clean design\n- **sculpture**: Artistic forms, museum-piece quality\n- **object**: Functional items, product design\n- **character**: Game figures, collectible characters\n\n### GET /api/jobs/{job_id}\nGet status and results for a specific job.\n\n### GET /api/jobs?agent_name=YourName\nList your jobs (or all jobs if no agent_name specified).\n\n### GET /api/stats\nGet service statistics and usage information.\n\n## Job Status Flow\n\n1. **pending** - Job submitted, waiting in queue\n2. **generating_image** - Creating 2D image from description\n3. **converting_3d** - Converting image to 3D mesh\n4. **estimating_cost** - Calculating printing costs\n5. **completed** - Job finished successfully\n6. **failed** - Job failed (check error_message)\n\n## Web
